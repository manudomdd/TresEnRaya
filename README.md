# Tres en Raya (Tic-Tac-Toe) en Java con Swing

Este proyecto es una implementación del clásico juego **Tres en Raya (Tic-Tac-Toe)** utilizando **Java** y la biblioteca gráfica **Swing**. El juego permite a dos jugadores interactuar en una interfaz gráfica de usuario (GUI) para jugar de forma local.

## Descripción

El juego sigue las reglas tradicionales del **Tres en Raya**, donde dos jugadores, X y O, se turnan para marcar las casillas de una cuadrícula de 3x3. El primer jugador que consiga alinear tres símbolos (ya sea en fila, columna o diagonal) gana la partida. Si la cuadrícula se llena sin que ninguno haya ganado, el juego termina en empate.

## Características

- **Interfaz Gráfica de Usuario (GUI)** creada con **Java Swing**.
- Jugabilidad de dos jugadores en el mismo dispositivo.
- Indicación visual de quien está jugando (X o O).
- Detección de victoria o empate.
- Opción para reiniciar el juego al finalizar una partida.

## Tecnologías

- **Lenguaje**: Java
- **Biblioteca de UI**: Java Swing
- **IDE**: NetBeans

## Requisitos

- **Java Development Kit (JDK)** versión 8 o superior.
- **NetBeans IDE** (opcional, pero recomendado para facilitar la gestión del proyecto).
